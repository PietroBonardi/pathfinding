### Overview
An interactive interface for visualizing shortest path algorithms. 
- set your starting point 📌
- add obstacles 👾
- set a destination 🏠

Then execute **to visualize the path**.

### Installation
- Clone the repository
```bash
git clone https://github.com/PietroBonardi/pathfinding
```
- Add dependecies
```bash
make local
```
- Run application: 
```bash
make run
```
